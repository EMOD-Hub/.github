# Security Policy

## Maintenance Status

This project is provided as open source software for community use and development under the MIT license.

The project is not actively maintained or supported by IDM & Gates Foundation.

Users and deployers are responsible for:
- evaluating the security of the software
- maintaining their own deployments
- applying patches or mitigations as needed
- assessing suitability for their environment

Security updates, vulnerability responses, and maintenance are not guaranteed.

## Reporting Vulnerabilities

This project does not currently provide a formal security response process.

Please use GitHub issues or community discussion channels for public discussion
of bugs or vulnerabilities, unless a maintainer has indicated otherwise.

## No Warranty

This software is provided under the terms of the project's open source license
without warranty of any kind.
