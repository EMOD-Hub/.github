# Key EMOD Repositories

EMOD-Hub contains a number of repositories supporting disease modeling, workflows,
execution platforms, tutorials, and supporting tools. The repositories below are
the ones most users are likely to interact with.

* **[EMOD](https://github.com/EMOD-Hub/EMOD)**
  The core disease modeling engine implemented in C++.

* **[emodpy-malaria](https://github.com/EMOD-Hub/emodpy-malaria)**
  Python workflows, utilities, tutorials, and tooling for EMOD-Malaria users.

* **[emodpy-hiv](https://github.com/EMOD-Hub/emodpy-hiv)**
  Python workflows, utilities, tutorials, and tooling for EMOD-HIV users.

* **[emodpy-workflow](https://github.com/EMOD-Hub/emodpy-workflow)**
  Higher-level workflow tooling used by some EMOD-HIV workflows and users migrating
  from DtkTools.

* **[emodpy](https://github.com/EMOD-Hub/emodpy)**
  Shared Python workflow infrastructure used by disease-specific EMOD packages.

* **[emod-api](https://github.com/EMOD-Hub/emod-api)**
  Python tools for working with EMOD configuration, campaign, and output files.
